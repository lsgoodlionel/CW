# 官方财务报表说明(小企业会计准则)

系统按《小企业会计准则》官方模板生成三张报表,并可一键导出税务报送用 Excel:

| 报表 | 表号 | 模板来源 |
|---|---|---|
| 资产负债表 | 会小企01表 | `docs/财务报表报送与信息采集（小企业会计准则）…` |
| 利润表 | 会小企02表 | 同上 |
| 现金流量表 | 会小企03表 | 同上 |

## 报表类型与列语义

| 类型 | 利润表/现金流量表 列1 | 列2 | 资产负债表 |
|---|---|---|---|
| 月报 | 本期金额(当月) | 本年累计金额 | 期末余额 / 年初余额 |
| 季报 | 本期金额(当季) | 本年累计金额 | 期末余额 / 年初余额 |
| 年报 | 本年累计金额(全年) | 上年金额 | 期末余额 / 年初余额 |

- 年初余额 = 上年 12-31 的累计余额
- 上年金额 = 上一年度全年发生额

## 科目 → 报表行次映射

映射定义见 `backend/app/reports_cn.py`:
- 资产负债表:`_asset_lines` / `_right_lines`(53 行)
- 利润表:`_income_values`(32 行)
- 现金流量表:`_cashflow_values`(22 行)

「未分配利润」= 利润分配/本年利润科目余额 + 损益类科目累计净额,保证「资产 = 负债 + 所有者权益」恒等。

## 现金流量表的推导方式(重要)

小企业记账系统通常不单独核算现金流量。本系统采用**对方科目归类法**近似推导:

1. 现金科目 = 库存资金(1001)、银行存款(1002)、其他货币资金(1012)
2. 遍历区间内每一张涉及现金科目的凭证,按现金增减方向,将其**对方科目**归入经营/投资/筹资各行
3. 未识别的对方科目:流入计入「收到其他与经营活动有关的现金」,流出计入「支付其他与经营活动有关的现金」
4. **现金净增加额 / 期初 / 期末余额直接取现金科目实际余额**,保证准确

> 因此各活动明细为近似分类,但期初、期末、净增加额与账面现金完全一致。
> 如需精确的现金流量分类,可在 `_CF_INFLOW` / `_CF_OUTFLOW` 中调整科目映射。

## API

```
GET /api/reports/official?report_type=month|quarter|year&year=&month=&quarter=
    → 三张报表结构化数据(页面展示)

GET /api/reports/export-excel?report_type=...&year=...&month=...&quarter=...
    → 按会小企模板导出 .xlsx(三个 sheet)
```
